## Designed for the DC Heroes 3rd edition TTRPG

### Install Dependencies

```bash
$ cd dc-heroes-tool

# using yarn or npm
$ yarn (or `npm install`)

# using pnpm
$ pnpm install --shamefully-hoist
```

### Use it

```bash
# development mode
$ yarn dev (or `npm run dev` or `pnpm run dev`)

# production build
$ yarn build:win64 (or `npm run build:win64` or `pnpm run build:win64`)
```
